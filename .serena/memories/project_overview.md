# AST 实时语音转文本与大模型分析系统

## 项目概述

AST是一个基于Python的实时语音转文本（ASR）与大模型分析系统，集成了语音识别、说话人识别（声纹）、语音活动检测（VAD）等技术，并提供Web界面支持实时交互。

### 核心功能

1. **实时语音转文本（ASR）**: 基于SenseVoice Small模型，支持多语言识别
2. **说话人识别**: 基于CAM++模型，支持声纹库管理和实时说话人识别
3. **语音活动检测（VAD）**: 基于WebRTC VAD，准确检测语音片段
4. **大模型对话**: 集成多种LLM API（OpenAI兼容），支持流式对话
5. **智能分析助手**: 基于底层小模型Agent，智能判定是否需要启动智囊团
6. **智囊团**: 多个不同LLM同时给出提议，辅助主人公进行语音讨论
7. **Web界面**: 响应式界面，支持实时显示转录结果和LLM对话
8. **多会话管理**: 支持聊天历史、会话切换等功能

### 技术特点

- **实时性**: 基于PyAudio实现实时音频采集和处理
- **多模型支持**: 支持OpenAI兼容的所有LLM API
- **声纹识别**: 自动识别说话人身份，支持声纹库管理
- **智能触发**: 基于字数累积和静音检测的智能分析机制
- **流式通信**: 使用WebSocket实现前后端实时通信
- **模块化设计**: 各功能模块独立，易于维护和扩展

### 应用场景

- 语音会议记录和分析
- 多说话人语音内容整理
- 语音讨论的AI辅助分析
- 实时语音翻译和转录
- 语音数据挖掘和分析

## 主要技术特性

- **智能分析触发机制**: 字数累积达到10字时激活判定，静音2秒时触发分析
- **三阶段智能分析**: 阶段1（小模型判定）→ 阶段2（意图识别）→ 阶段3（分发准备）
- **多模型协作**: 支持智囊团模式，多个LLM同时给出专业建议
- **流式响应**: 前端实时显示转录结果和AI回复
- **声纹库管理**: 支持用户自定义声纹录入和识别
